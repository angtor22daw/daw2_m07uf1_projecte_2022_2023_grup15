Informació del lloc web


Aquest lloc web permet autenticar-se per rols, un rol administrador o un rol d'usuari basic. Els usuaris administradors poden veure els alumnes i les seves notes, crear alumnes i usuaris que poden autenticarse, modificar notes dels alumnes i eliminar alumnes. Els usuaris basics només poden veure els alumnes i les seves notes. Tot aquest lloc web està fet amb PHP, HTML, CSS i Bootstrap.
